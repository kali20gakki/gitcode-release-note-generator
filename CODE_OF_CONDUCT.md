# 行为准则

本项目致力于提供一个友好、包容和 harassment-free 的参与环境。

## 我们的标准

有助于创造积极环境的行为：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为：

- 使用性化语言或图像，以及不受欢迎的性关注或 advances
- 发表挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中被合理认定为不恰当的行为

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时的行为。

## 执行

可以通过创建 Issue 联系项目维护者报告滥用、骚扰或其他不可接受的行为。
